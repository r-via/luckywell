# LuckyWell

> Before you ape in, throw a coin.

LuckyWell est un jeu rituel on-chain construit sur Solana. Les traders jettent un coin dans le puits, reçoivent une prédiction de mindset, et ont une chance de gagner 2x à 5x leur mise.

## Stack technique

- **Frontend** : [Vike](https://vike.dev) (SSR / SSG)
- **Blockchain** : Solana
- **Style** : CSS vanilla avec variables custom
- **Fonts** : Fredoka, Nunito, Space Mono

## Fonctionnalités

- Lancer un coin dans le puits on-chain
- Recevoir une prédiction axée trading (mindset)
- Gagner entre 2x et 5x avec 28% de chances
- Odds transparentes et identiques pour tous
- Interface bilingue (EN / FR)

## Structure du projet

```
luckywell.html          # Application principale (single-page)
luckywell.code-workspace # Configuration VS Code
README.md               # Documentation
```

## Développement

Le projet utilise Vike comme framework de rendu. Pour démarrer :

```bash
npm install
npm run dev
```

## Philosophie

LuckyWell n'est pas un bot de trading, pas un conseil financier, et ne promet aucun profit. C'est un rituel transparent et on-chain pour aider les traders à entrer sur le marché avec intention.
